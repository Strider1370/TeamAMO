# 11. 네이티브/모바일 기능 — "반응형 ≠ 앱" 확장 자산

> 목적: 폰 기능으로 "앱다움"을 만든다. **코어(텍스트→슬롯→규칙→타임라인)는 P0로 불변**, 네이티브는 그 위 레이어.
> 결정(2026-06-23, 사용자 승인): GPS·가까운기관지도·카메라서류·전화·이미지공유 채택 / 푸시는 시연용 껍데기.
> 대부분 **모바일 웹 브라우저 API로 동작**(APK 불필요). 풀 네이티브는 reference 05(Capacitor APK) 경로.

## 채택 기능 6종

### A. 전화 연결 (P0 — 거의 무료)
- curated-benefits의 `phone` 필드(1577-1000·1899-9988·129…) → 결과 카드에 `tel:` 링크. 탭 → 다이얼.
- 카카오 기관검색 결과 전화도 동일. **데이터 준비 완료, 코어에 포함.**

### B. GPS 지역 자동선택 (P1)
- 브라우저 `navigator.geolocation` → 좌표 → 카카오 로컬 역지오코딩 → 시군구 자동 → 지자체 제도 필터(수동 지역선택 제거).

### C. 가까운 기관 지도/길찾기 (P1) — 카카오맵
- **핵심 트릭**: `facilities.json`은 주소만(좌표無) → 직접 지도 불가. 대신 **카카오 로컬 키워드/카테고리 검색**으로 GPS 좌표 근처 "치매안심센터·행정복지센터·노인복지관" 검색 → 좌표·거리·전화 반환 → 카카오맵 마커 + 길찾기 링크. **우리가 좌표 데이터를 안 만들어도 됨.**
- 의존성: 카카오 개발자 키 **JS 키(지도) + REST 키(검색)** + 도메인 등록(localhost·배포). 사용자 발급(미리). 키는 런타임/env, 깃 ❌. REST 키는 서버사이드.

### D. 카메라 서류 입력 (P1)
- 촬영: 모바일 웹 `<input type="file" accept="image/*" capture="environment">` (무료).
- 해석: **GPT-5.5 비전**으로 사진→상황 추출(별도 OCR 불필요, 같은 LLM 키 재사용). 진단서·퇴원안내문·고지서 → 슬롯.
- 정직성: 추출 결과를 "이해 중" 슬롯 칩으로 보여주고 사용자 확인(오독 방지).

### E. 이미지 저장/공유 (P1)
- "오늘 할 일" 결과 → PNG 저장(html2canvas/dom-to-image, CDN) + `navigator.share`(Web Share, 네이티브 공유 시트).
- 용도: 가족 공유(형제자매 역할분담)·본인 기록 보관. 가상 페르소나 시연 → 개인정보 이슈 없음.

### F. 푸시 기한·순서 알림 (P2 — 시연용 껍데기)
- 진짜 푸시는 서비스워커+백엔드 스케줄링 → 4시간 무리. **UI 목업만**: "알림 미리보기" 화면(예: `재난적의료비 마감 D-30`, `이제 복지용구 신청할 때예요`).
- ⚠️ **정직성 필수**: 발표에서 "작동합니다" ❌ → **"이렇게 알려드릴 예정"** 시연 목업으로 명확히. (curated의 deadline·sequence 데이터가 이 알림의 근거.)

## LLM (확정)
- **OpenAI GPT-5.5**(2026-04 플래그십, 텍스트+이미지 멀티모달). 자연어 슬롯 추출 + 카메라 서류 비전 + 쉬운말 번역 한 모델로.
- 키: **앱 런타임 입력(설정 ⚙️ 모달), env, 깃 ❌.** 라이브 데모 지연 시 빠른 mini(gpt-5-mini 등)로 폴백 옵션. 키 없으면 slot-dictionary 키워드 매칭 폴백.
- 빌드 배선: web/ 스캐폴드 `lib/llm.ts`가 OpenAI 전용이라 모델 id만 교체(`gpt-4.1-mini`→`gpt-5.5`), 비전 입력 경로 추가.

## 범위 순서 (요약)
- **P0**: 코어 엔진 + 전화연결 + (검토 5구멍)
- **P1**: GPS지역자동 · 카카오 지도/길찾기 · 카메라 서류 · 이미지 저장/공유
- **P2**: 푸시 껍데기(정직 프레이밍)
- 코어 먼저, P1은 시간 되는 순서대로, 못 넣는 건 **확장 슬라이드**(데이터·디자인 다 있어 진짜로 보임).

## 발표 내러티브 (확장 1장)
> "막막함을 *오늘 할 일*로 바꾸고 — 폰에 담기면 **사진 한 장으로 입력**하고, **위치로 길을 찾고**, **전화 한 번으로 연결**하고, **가족과 공유**하고, **때가 되면 먼저 알려주는** 동반자가 됩니다." (같은 엔진 + 폰 기능)
