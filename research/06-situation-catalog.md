## 0. 사용 가이드 (빌드팀용)

- 각 제도 행의 `공식 URL`은 우선순위: ① 소관기관 공식 페이지 → ② 복지로(bokjiro) 상세 → ③ 정부24(gov.kr) → ④ 찾기쉬운 생활법령(easylaw). 룰 데이터에는 가능하면 소관기관 1차 링크를 박을 것.

## Task 1 — 추가 상황 분류 발굴 (기존 6개가 놓친 것)

기존 6개 분류(은퇴/입원/퇴원후ADL/치매노쇠/가족환경변화/경제주거)는 "생존 중 돌봄"에 집중되어 있다. 실재 확인된 제도가 있는 것만 채택했다.

### 추가 분류 A — 말기·임종·사후행정 ✅채택
- **왜 필요한가:** 6개 분류 어디에도 "죽음의 과정과 그 직후 행정"이 없다. 자녀가 가장 막막해하는 순간(임종 임박, 사망 직후 연금·상속·장례)이 공백.
- **겹침:** 분류2(입원/암)와 의료 측면에서 일부 겹치나, 호스피스·연명의료·장제급여·유족연금 승계는 별도 트랙. 분류5(배우자사망→유족연금)와 유족연금에서 겹침 → 룰에서 두 분류 모두 점등시키면 됨.
- **확인된 실재 제도:** 호스피스·완화의료, 사전연명의료의향서, 장제급여(기초생활보장), 유족연금(국민연금), 사망일시금.

### 추가 분류 B — 노인성 질환 장애등록 경로 ✅채택
- **왜 필요한가:** 뇌졸중·파킨슨·치매 후유증으로 거동/인지가 영구 저하되면 "장기요양"만으론 부족하고 **장애인 등록**이 별도 혜택문(장애인연금·활동지원·각종 감면)을 연다. 자녀가 "장기요양=장애등록"으로 오해하기 쉬워 안내 필요.
- **겹침:** 분류3(퇴원후 ADL)·분류4(노쇠)와 신체기능 측면 겹침. 단 장애등록은 의료적 장애판정 트랙으로 별도. 장기요양과 장애인활동지원은 원칙적으로 중복 수급 제한 → 룰에서 "양자택일/조정 안내" 플래그 필요.
- **확인된 실재 제도:** 장애인 등록(시군구·국민연금공단 심사), 장애인연금, 장애인활동지원.

### 추가 분류 C — 권리·안전 보호 ✅채택
- **왜 필요한가:** 독거·치매 노인의 **학대·금융착취·판단능력 상실** 상황. 응급안전·학대신고·성년후견은 "돌봄"이 아니라 "권리방어" 트랙이라 6개에 안 잡힘.
- **겹침:** 분류4(치매→금융사기예방)·분류5(독거→안부확인)와 부분 겹침. 응급안전안심서비스는 분류5와 강하게 겹치므로 룰에서 공유.
- **확인된 실재 제도:** 독거노인·장애인 응급안전안심서비스, 노인보호전문기관(노인학대 신고 1577-1389), 성년후견제도(법원).

### 추가 분류 D — 돌봄자(자녀) 본인 지원 ✅채택
- **왜 필요한가:** 이 서비스의 사용자는 "자녀". 부모 제도만 안내하면 보호자 소진을 놓침. 자녀 본인이 쓸 수 있는 휴직·휴가·단기보호(레스핏)가 별도 트랙.
- **겹침:** 분류2·분류5에 "가족돌봄휴가·휴직"이 이미 언급됨 → 이 분류는 그걸 **자녀 관점으로 재집결**시키는 횡단(cross-cutting) 분류. 치매가족휴가제(장기요양 가족휴가제)는 분류4와 겹침.
- **확인된 실재 제도:** 가족돌봄휴직·가족돌봄휴가(남녀고용평등법), 가족돌봄등근로시간단축, 장기요양 가족휴가제(=치매가족휴가제, 단기보호).

### (참고) 특수대상 교차필터 — 분류 아님, **필터 축**으로 처리 권장
- 국가유공자·보훈 대상, 농어촌(농지연금) 등은 "상황 분류"가 아니라 **사용자 속성 필터**. 룰 엔진에서 별도 태그축(`tag:보훈`, `tag:농어촌`)으로 두고 해당 시 추가 제도를 가산하는 구조 권장. 농지연금만 아래 카탈로그에 1행 수록.

## Task 2 심화 — 신청 절차·서류·순서 (핵심 8제도 한 단계 더)

### ① 노인장기요양보험 장기요양인정신청
- **자격:** 건강보험 가입자/피부양자 또는 의료급여수급권자 중, (a) 만 65세 이상 또는 (b) 만 65세 미만이면서 치매·뇌혈관질환 등 노인성 질병 보유.
- **준비서류:** 장기요양인정신청서 + 의사소견서. (65세 미만은 노인성 질병 상병코드가 명시된 진단서/의사소견서를 접수 시점부터 함께 제출해야 정상 심사.)
- **순서:** ① 신청(공단 운영센터 방문·우편·팩스·인터넷·The건강보험앱) → ② 공단 직원 방문 인정조사 + 의사소견서 → ③ 등급판정위원회 등급 판정(1~5등급, 인지지원등급) → ④ 공단이 장기요양인정서 + 개인별장기요양이용계획서 송부 → ⑤ 장기요양기관과 계약, 급여 이용.
- 공식: https://www.longtermcare.or.kr

### ② 치매안심센터 등록
- **자격:** 만 60세 이상은 신분증만으로 무료 선별검진 가능. 진단/감별검사비 지원은 중위 120% 이하.
- **순서:** ① 관할 치매안심센터 방문(신분증) → ② 1단계 선별검사(인지선별) → ③ 인지저하 시 거점병원 진단·감별검사(검사비 지원) → ④ 치매 진단 시 센터 등록관리(사례관리·치료관리비·인식표·쉼터 등 연계).
- 검사비 지원 예: 진단검사 최대 15만원, 감별검사 병의원급 8만원·상급종합 11만원.
- 공식: https://www.mohw.go.kr/menu.es?mid=a10712010100 / https://ansim.nid.or.kr

### ③ 노인맞춤돌봄서비스
- **자격:** 만 65세 이상 기초수급/차상위/기초연금 수급자 중 돌봄필요(독거·조손·고령부부·기능저하·우울 등). **장기요양·가사간병·보훈재가·장애인활동지원 등 유사 재가서비스 수급자는 제외**(중복 불가).
- **서류:** 신분증, 주민등록등본, (해당 시) 기초수급/차상위 증명서.
- **순서:** ① 읍면동 방문 또는 복지로 신청 → ② 대상자 선정조사(욕구·상태) → ③ 서비스 군 배정 → ④ 수행기관 매칭, 안전확인·가사·외출동행 등 제공(무료).

### ④ 기초연금

- **순서:** ① 만 65세 생일 1개월 전부터 신청 가능 → ② 읍면동/국민연금공단지사/복지로 신청 → ③ 소득·재산 조사 → ④ 결정통지 후 매월 지급. (부부 동시 수급 시 감액, 공무원연금 등 일부 직역연금 수급자 제외 규정 있음 — 재확인.)

### 다축 동시 점등 예시 (AI 다축 해석 근거)

- **예시 A — "치매 + 독거 + 저소득":**
  점등 = {치매안심센터, 장기요양인정, 노인맞춤돌봄, 응급안전안심서비스, 성년후견(판단능력 저하 시), 기초생활보장/의료급여, 에너지바우처}.
  → 분류2(치매), 분류C(권리·안전), 분류5(독거), 분류6(저소득)이 동시 점등. 룰 충돌 체크: 노인맞춤돌봄은 장기요양과 중복 불가 → "장기요양 우선/노인맞춤 제외" 가지치기 필요.

- **예시 B — "뇌졸중 입원 + 퇴원 후 마비 + 보호자(자녀) 직장인":**
  점등 = {재난적의료비, 산정특례, 간호간병통합(입원기), → 퇴원 후: 장기요양인정, 방문요양, 복지용구, → 영구마비: 장애인 등록·장애인연금·활동지원, → 자녀: 가족돌봄휴직·근로시간단축}.
  → 분류2 → 분류3 → 분류B → 분류D로 **시간축 전이**까지 표현. 체크리스트를 "지금/퇴원 후/장기" 타임라인으로 분할하는 근거.

- **예시 C — "배우자 사망 + 남은 부모 독거 + 우울":**
  점등 = {유족연금, 장제급여(수급자), 가구원 변경 복지 재점검, 노인맞춤돌봄(정서지원), 응급안전안심, (위기 심화 시)긴급복지}.
  → 분류A(사후행정) + 분류5(독거/사별) + 분류C(안전) 동시. "한 사건(사망)이 연금·행정·돌봄 3트랙을 동시에 연다"는 내비게이터 킥의 핵심 사례.
